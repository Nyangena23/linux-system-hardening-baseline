# Linux System Hardening Baseline

## Objective
Demonstrate hands-on Linux security hardening aligned with entry-level SOC and IT roles.

## Environment
- OS: Ubuntu 22.04 LTS
- Tools: Linux CLI, UFW, SSH

## Hardening Measures
- User privilege separation
- SSH configuration hardening
- Firewall enforcement
- Password policy enforcement
- Service minimization

Before and after system states documented in /verification.

## Outcome
Improved system security posture and reduced exposure to common attack vectors.
